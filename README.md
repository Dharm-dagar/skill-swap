# 💡 Skill Swap Platform

A mini web application that enables users to list their skills and request other skills in return.


## 🚀 Features

- 🔐 User authentication (signup/login)
- 📜 Add and manage your skills
- 🔍 Browse and search for available skills from others
- 🤝 Send and receive skill swap requests
- 📬 Notification system for pending/existing requests
- 💬 Optional messaging/chat integration
